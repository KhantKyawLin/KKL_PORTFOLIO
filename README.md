# 🚀 Personal Portfolio & Digital CV

Welcome to the repository for my personal portfolio website! This project is a reflection of my journey from healthcare and education into the world of Full-Stack Development. It is designed to be **simple, useful, and secure**, highlighting my technical expertise and pragmatic approach to problem-solving.

## 🌐 Live Demo
Check out the live version here: **[kkl-portfolio.vercel.app](https://kkl-portfolio.vercel.app/)**

## ✨ Key Features
- **Modern & Responsive Design:** Built using Bootstrap 5 for a seamless experience across all devices.
- **Dark/Light Mode:** Integrated theme toggle that even remembers your preference.
- **Dynamic Projects Showcase:** Highlighting my work in Laravel, Flutter, and PHP.
- **Interactive Skills Slider:** Visually engaging display of my technical stack using Devicons.
- **Digital CV:** A professional, single-page printable CV integrated directly into the site.
- **Contact Form:** Fully functional form handling via FormSubmit.co.
- **Animations:** Smooth scroll and reveal effects using AOS (Animate On Scroll) and Typed.js.

## 🛠 Tech Stack
-   **Frontend:** HTML5, CSS3, JavaScript (ES6+), Bootstrap 5, Font Awesome, Devicon
-   **Animations & UI:** AOS.js, Typed.js, Swiper.js, SweetAlert2
-   **Backend/Services:** FormSubmit.co (Contact Form handling)
-   **Deployment:** Vercel
